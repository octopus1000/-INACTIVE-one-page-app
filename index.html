How this app come out?

The scence for this page is inside an elevator.The reason I set up such a scene is for switching between different sections.

I used to use css 3d features(rotate and perspective) to construct the elevator, yet the divs shake when animation trasition performed(when on chrome). I thought about and found that I don't really need a 3D elevator, so I draw on a 2-D canvas instead.
